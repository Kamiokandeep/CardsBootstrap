Description
Bootstrap Cards

The Bootstrap Cards Component is a versatile tool that allows you to create visually appealing and customizable content boxes. These cards are containers for information that can include text, images, links, buttons, and other HTML elements. The default design of the cards is clean, modern, and fully responsive, automatically adapting to different screen sizes.

Bootstrap.com
Common uses in a Web Application
  • Content Presentation
  • Image Galleries
  • Product or Service Listings
  • Profile or User Components
  • Modular Design
  • User Interactions

Cards are very common on websites because they help organize news in a very dynamic way, for example this is the BBC news page. They can also have a title, an image, a description of the news and a publication date. Cards also help organize news sections more easily in mobile versions.

Explanation of Classes Used:
  • card: Creates a flexible and extensible content container with multiple variants and options.
  • card-img-top: Adds an image to the top of the card, keeping it responsive.
  • card-body: Defines the main content area of the card.
  • card-title: Styles the title within the card body.
  • card-text: Styles regular text inside the card body.
  • btn: Applies default button styles to an anchor element.
  • btn-primary: Adds a primary color to the button (based on the Bootstrap theme).
  • bg-light: Provides a light gray background for better visibility.
  • p-3: Adds padding (1rem) around the content for spacing.
  • rounded: Gives rounded corners to the container.

Explanation of how to implement a Bootstrap card:

  • Prepare the Environment: Before using Bootstrap components, make sure that the Bootstrap CSS and JavaScript files are properly linked in your project.

    Link to Bootstrap JavaScript: <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous" script>


  • Structure: Create a basic card with the main elements: image, header, text and button. 
                                                    
<div class="card" style="width: 18rem;">
    <img src="https://via.placeholder.com/286x180" class="card-img-top" alt="Example Image">
    <div class="card-body">
        <h5 class="card-title">Card Title</h5>
        <p class="card-text">This is a brief description of the card content.</p>
        <a href="#" class="btn btn-primary">Go somewhere</a>
    </div>
</div>
  • Component Customization: Use CSS to customize the appearance. For example: 
                                                    
.card {
    border: 2px solid #8d29eb; /* Change the border */
    border-radius: 10px;       /* Rounded edges */
}

.card-title {
    color: #9d29eb;            /* Change title color */
}
